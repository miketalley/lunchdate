Lunch Date README

This software is developed to allow users to schedule lunch dates online and allow
other users to be able to view the date location and time and choose if they
would like to accept the date.

To install the software you can clone the repository. Some notes are that this
software uses {GMaps4Rails Gem}[https://github.com/apneadiving/Google-Maps-for-Rails],
and also the {Google Places Gem}[https://github.com/marceldegraaf/google_places].


* Ruby version
  2.1.1

* System dependencies
  Requires {Underscore.js}[http://underscorejs.org/]

* Database creation
  rake db:create

* Database initialization
  rake db:migrate
